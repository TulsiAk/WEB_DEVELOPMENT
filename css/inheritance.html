CSS Inheritance

Understanding Inheritance

CSS inheritance is a mechanism where certain properties of an element are passed down to its child elements.

For example, if you set a font size on a parent element, all child elements will inherit that font size unless they have their own specific font size set.

which can't be Inherited

Not all CSS properties are inheritable. Properties like button, input, margin, padding, and border do not inherit by default.

To make a property inherit, we can use the "inherit" value.
